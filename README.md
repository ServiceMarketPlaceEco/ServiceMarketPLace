
# ServiceHub Frontend

---

## Tech Stack

- Vue 3 (Composition API)
- Vite (build tool)
- JavaScript (ES6+)
- CSS (custom styling)
- Node.js & npm

---
---

## How to Run the Frontend

### 1. Install Node.js

Make sure Node.js is installed:

```bash
node -v
npm -v 
```
### 2. cd servicehub
### 3. npm install
### 4. npm run dev

Run Vite URL into your web browser
